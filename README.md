# Chat Bot

A flexible Twitch chatbot

## Setup

Requires Node v18.

```
yarn install
yarn start
```

## Built-in Commands
- Auto-shoutout
- `!followage` tells you how long you've been following
- `!anarchy`, `!x`, `!league`, `!regular` gives access to the splatoon3.ink map/mode rotation info
